# Kyverno101
Exploring Kyverno for K8s

This repository consists of documents pertaining to Kyverno and can be used as a practical guide to getting started with Kyverno — a policy engine for Kubernetes that allows you to define, enforce, and audit policies using plain YAML.  

In repository, it cotains the following files:  
Kyverno101 - Core Concepts.md -> Goes more in depth on the architecture and core concepts available in Kyverno  
Kyverno101 - setup.md -> Guide to install miniKube and Kyverno on Windows and WSL  
Kyverno101 - Validate Use Case.md -> Guide to constructing and testing a validation use case in Kyverno  
fix-minikube.sh -> Required file when setting up minKube  
only-allow-port-443.yaml -> File used in the Validation use case
