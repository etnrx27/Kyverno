# Kyverno
Kyverno is a policy engine for Kubernetes that allows one to define, enforce, and audit policies using YAML and CEL.    
This repository will be exploring the fundamentals and use cases of Kyverno. 

## Kyverno101
This repository consists of reference documents pertaining to Kyverno that gives any engineer a solid foundation in Kyverno.  
It covers Kyverno's Architecture and how Kyverno integrates with the Kubernetes API.  
It explores the various policy types Kyverno covers.   
It provides simple examples of policy implementation as a baseline. 

## Proof Of Concepts
This repository consists of some proof of concepts that showcases Kyverno capabilities.
### Replacing Openshift SCCs on Native Kubernetes using Kyverno
Showcasing that Kyverno policies can replicate the security controls currently provided by OpenShift SCCs and ACM-managed `NetworkPolicy` defaults.
